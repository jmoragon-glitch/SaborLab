# Team Agreements – Modo Individual

**Proyecto:** SaborLab  
**Desarrollador:** Jaime Mora  

---

## 1. Normas de trabajo
- Trabajo individual con autoevaluación continua.  
- Código revisado manualmente antes de cada commit.  
- Se utilizará GitKraken y GitHub como control de versiones.

---

## 2. Responsabilidades
- Planificación, diseño, desarrollo y documentación a cargo del mismo autor.  
- Revisión y validación de requerimientos con el cliente (profesor/a).

---

## 3. Control de calidad
**Checklist por tarea:**
- [x] Código legible y comentado.  
- [x] Cumple guías de estilo y accesibilidad.  
- [x] Validaciones probadas manualmente.  
- [x] Commit con mensaje claro según convención.  
- [x] README actualizado.

---

## 4. Seguimiento de contribuciones
- Historial de commits y changelog servirán como evidencia de avance.  
- Se mantendrá un registro semanal de hitos en `CHANGELOG.md`.

---

## 5. Valores personales
> Autonomía, claridad y mejora continua.  
> “Cada commit cuenta una historia de progreso.”
